# Weather Forecast Application

A weather forecasting application that provides real-time weather information based on the user's current location or a searched city.

## Features

- Detect current location
- Search weather by city
- Real-time weather information
- Dynamic API integration
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript(ES6+)
- Weather API
